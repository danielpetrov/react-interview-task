## Setup

#### node.js LTS, npm and github are required

`git clone https://github.com/danielpetrov/react-interview-task.git`

`npm install`

## Start development

`npm run start`

## Lint

`npm run lint`

## Stack - development

* axios
* immutable
* ramda
* react
* react-modal
* react-select
* react-notifications
* redux
* redux-saga
